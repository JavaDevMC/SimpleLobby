# SimpleLobby

SimpleLobby is a easy to setup lobby plugin for the minecraft version 1.8.


## Commands

```xml
/sl setspawn
/sl reload
/sl spawn
/spawn
/hub
/lobby
/stuck
/l
```

## Permissions

```xml
simplelobby.admin
simplelobby.cmd
```
